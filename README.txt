 Task 1: Python File Handling & Automation

 Objective
The objective of this task is to implement file handling in Python by reading student data from a file and generating an automated report.

 Description
This project demonstrates the use of Python file handling and basic automation. 
The program reads student records from an input file (student_data.txt), processes the data, and generates a well-structured report (report.txt). 
Additionally, a backup of the report is created (backup_report.txt) to ensure data safety.

 Features
- Reading data from a text file
- Processing student information (Name, Course, Marks)
- Generating a formatted report
- Creating a backup file automatically
- Simple and clean code structure

 Files Included
- task.py → Main Python script
- student_data.txt → Input file containing student details
- report.txt → Generated report file
- backup_report.txt → Backup of the report
- code.png → Screenshot of the code
- output.png → Screenshot of the output

 How to Run
1. Open the project folder in VS Code
2. Open the terminal
3. Run the following command:
   python task.py

 Output
The program generates:
- report.txt (final report)
- backup_report.txt (backup file)

Terminal Output:
Report Generated Successfully!

 Conclusion
This task helps in understanding Python file handling and demonstrates how automation can reduce manual effort in generating reports.
